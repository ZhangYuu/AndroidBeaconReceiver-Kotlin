# KotlinBeaconReceiver

Becaon Scanning and notifaction Demo application. This Demo app works on samsung S7 smart phone.It keeps scanning nearby beacon devices when identify certain Becaon UUID comes, app will make a notificatoin for certain beacon devices being detected.
